# 家計簿アプリ

Python製のシンプルなデスクトップ家計簿アプリです。

## 機能

- 収支の入力・記録（日付・種類・カテゴリ・金額・メモ）
- 月別の集計表示（収入・支出・差額）
- カテゴリ別支出の内訳表示

## 必要環境

- Python 3.8 以上
- tkinter（Python標準ライブラリ、追加インストール不要）

## 使い方

```bash
python main.py
```

データは `data.json` に自動保存されます。
